[]== MAKEFILES ==[]
uses the "make" utility, tells it what to make.

Example:
[]===========================[]
 |  all: hello               |
 |  hello: hello.c           |
 |    gcc -o hello hello.c   |
 |                           |
 |  clean:                   |
 |    rm -f *.o hello        |
[]===========================[]
